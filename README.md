# AI Movie Recommendation System

## Description

This project is an AI-based movie recommendation system that suggests movies based on user preferences using content-based filtering.
---
## Technologies Used

* Python
* Pandas
* Scikit-learn
---
## Project Structure
* `app.py` → main application
* `recommender.py` → recommendation logic
* `data_loader.py` → data handling
* `movies.csv`, `ratings.csv`, etc. → dataset
---
##  Features
* Recommend similar movies
* Uses machine learning techniques
* Fast and simple suggestions
---
## How to Run
### 1. Install dependencies
pip install pandas scikit-learn
### 2. Run the project
python app.py
---
## Note
Make sure all dataset files are in the same directory.
